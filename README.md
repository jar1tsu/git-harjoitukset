# Hello-sovellus

**Kurssi:** Git-versionhallinta (SOF013As2A-3001)
**Tekijä:** Jari Kuusikko

## Kuvaus

Tämä repositorio sisältää kurssin harjoituksissa tehdyn sovelluksen.

Projektissa on harjoiteltu:
- Gitin peruskäyttöä (init, add, commit, log, revert, reset, checkout/switch)
- Haarojen (branchien) käyttöä ja yhdistämistä (`develop`, feature-haarat)
- Etärepositorion (GitHub) käyttöä, push/pull, sekä pull requesteja
- Yksinkertaisen HTML/CSS/JavaScript-sovelluksen kehittämistä:
  - `index.html`: Hello-sivu ja kellonäkymä
  - `styles.css`: sivun ja kellonäkymän tyylit
  - `scripts.js`: kellonajan jatkuva näyttäminen käyttäjälle
